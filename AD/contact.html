ADAS-Viewer can be opened with most types of web browsers (i.e., as below) on Windows, Linux, and Mac OS. As long as your OS has been installed the latest version of any web browsers, you can freely use ADAS-viewer without a technical issue. There is a list of web browsers and their versions we tested and compatible.

Environment

ADAS-Viewer are compatible with any of web browsers and the version for Windows, Linux, and Mac as follows:

• Chrome version 56.02924.87 (64-bit)
• Firefox version 52.0.2 (64-bit)
• Internet Explorer 11.953.14393.0 (64-bit)
• Microsoft Edge 38.14393.0.0 (64-bit)
• Safari 10.1 (64-bit)

Please address technical issues, problems, and questions about ADAS-viewer by sending an email to issue.leelab@gmail.com.
